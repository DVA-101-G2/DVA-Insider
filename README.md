DVA Insider
===================
En community sida för datavetenskapliga programmet som ska stärka förhållandet mellan elever och näringslivet.

Använder
--------------------
* PHP med http://codeigniter.com (2.1.2)
* MySQL
* JQuery

All dokumentation sker på svenska.

--------------------

--------------------
